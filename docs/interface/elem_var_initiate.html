private subroutine elem_var_initiate(Obj, tprop, tpoint, telem, names)

type(String), intent(in), optional :: names(:)

Description

This subroutine initiates QuadratureVariables_ object - If colID=M then object stores material properties - In above case tpoint is not considered and set to 1 internally - If names are present then size( names ) == tprop - If names are not present then variables will be named as V1, V2,...
