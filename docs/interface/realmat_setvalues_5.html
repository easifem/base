Introduction

This subroutine set values in obj%l - If ExtraOption=MATRIX_DIAGONAL then diagonal values are set; and Indx denotes the diagonal number with 0 being the main diagonal - If Extraoption=ROW then row values are set; Indx then denotes row number - If Extraoption=COLUMN then col values are set; Indx then denotes col number
