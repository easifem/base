Sparsekit_

type, public, EXTENDS( LinSolver_ ) :: Sparsekit_

Sparsekit_ data type is a container around Yusef Saad’s SparseKit lib.

  • Reference : https://www-users.cs.umn.edu/~saad/software/SPARSKIT/
  • This class interface sparsekit and EASIFEM
